      The Schema Scraper class.

      The Schema Scraper class fetches and processes all the schema's from a given URL and is used to provide all the schema's available instead of an hardcoded subsection.

      author Derk-Jan Karrenbeld <derk-jan+schema@karrenbeld.info>
      version 1.0
      package WordPress\Derk-Jan\Schema-Creator\Scraper

      singleton() : \DJ_SchemaScraper
      APIStatic

      DJ_SchemaScraper::singleton() will always return the same instance during a PHP processing stack. This way actions will not be queued duplicately and caching of processed values is not neccesary.
