      The schema viewer class.

      This add in builds a schema.org copy of the website from the schema data provided from cache or the Scraper. It also provided quick access for starring and enabling/disabling properties for certain classes.

      author Derk-Jan Karrenbeld <derk-jan+schema@karrenbeld.info>
      version 1.0
      package WordPress\Derk-Jan\Schema-Creator\Viewer

      singleton() : \the
      APIStatic

      DJ_SchemaViewer::singleton() will always return the same instance during a PHP processing stack. This way actions will not be queued duplicately and caching of processed values is not neccesary.
