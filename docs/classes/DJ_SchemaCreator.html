    Gets a singleton of this class

    singleton() : \DJ_SchemaCreator
    APIStatic

    DJ_SchemaCreator::singleton() will always return the same instance during a PHP processing stack. This way actions will not be queued duplicately and caching of processed values is not neccesary.
